  7. {% trans 'View Map' %}

    {% blocktrans %} On clicking the View Map button DataPortal is directed to this page - an interactive mapping section. DataPortal gives the user the ability to plot information on a map, making the data easier to visualise and sort.{% endblocktrans %}

    1. {% trans 'Using the Map' %}

      {% trans 'The interface:'%}

      {% blocktrans %}This is the starting page where a user can insert data onto a map to be analysed further. The familiar sidebar contains the dashboard button, options for importing data onto the map using layers, and Recent Search Results which is a quick way of adding previously used data. The layer button (located in the top right hand corner of the map, allows the user to select layers to view or hide.{% endblocktrans %}

    2. {% trans 'Adding data to the Map' %}

      {% blocktrans %}The WISERD DataPortal can be used to find and map a variety of socio-economic data, points of interest and boundary data for Wales. You can then export the map and download the data for further analysis.{% endblocktrans %}

      1. {% trans 'Remote Data' %}

        {% blocktrans %}To add remote data to the map, click on the Map Layers button and select Remote Data. {% endblocktrans %}

        {% blocktrans %}The following search box will appear, by typing in a key word different datasets will appear. Select any dataset of interest and click Add to Map.{% endblocktrans %}

        {% blocktrans %}A list of variables will appear, these help divide the data into meaningful layers, for example, choosing Very Good Health, Annually as a Percent in Total by LSOA. Click {% endblocktrans %}{% trans 'Render to Map.'%} {% blocktrans %} One common error that may occur is shown below, if you see this error it means you haven't selected enough variables. The error box will list all the variables that have not been chosen to help you select the correct ones.{% endblocktrans %}

        {% trans 'The following box will show'%} {% trans 'Choropleth Options'%} {% trans '(these are the visual features of the layer) select the desired options and click'%} {% trans 'Show New Layer.'%}

        {% blocktrans %}The following will be displayed. The key at the bottom of the map shows the different percentages of very good health in different regions.{% endblocktrans %}

         

        {% blocktrans %}From this map you can select a region and find out information about it in the righthand sidebar (example shown below).{% endblocktrans %}

      2. {% trans 'Local Data'%}

        {% blocktrans %}The WISERD DataPortal allows users to map their own data for areas in Wales and to export this as a publishable map. This function has been specially designed so that all the mapping occurs inside the users browser rather than on the WISERD DataPortal server. This means that potentially sensitive data does not leave the users machine.{% endblocktrans %}

        {% trans 'You can import your own data onto the DataPortal by clicking the MapMyData button.'%}

        {% blocktrans %}A box will appear where you can choose a local file to upload onto the map. The following box will display the fields available to display on the map.{% endblocktrans %}

        {% blocktrans %}You must select the Choropleth options so that your data can be displayed correctly and name the layer. Once {% endblocktrans %} {% trans 'Show New Layer' %} {% blocktrans %}is clicked the data will be displayed on the map including a key on the righthand side. {% endblocktrans %}

        {% blocktrans %}This will remain on the map until you choose to delete it. To turn the layer off use the {% endblocktrans %} {% trans 'Layer Manager'%} {% trans '(the brown icon in the top right hand corner of the map).'%}
